# Fork of the March Lunchlist made by @Ortix92

## Setup
Setup WAMP (kinda annoying but you can do it)  
Clone the project with PHPstorm into your WAMP directory.  
Run `composer install`  
Add a database file at `database/database.sqlite`  
Rename `.env.example` to `.env`
